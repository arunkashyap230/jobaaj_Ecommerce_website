# MERN E-Commerce Website

A full-stack E-Commerce web application built using the MERN Stack with secure authentication, product management, cart system, Razorpay payment integration, and Cloudinary image upload support.

---

# 🚀 Features

- User Authentication (Login / Signup)
- JWT Authentication & Authorization
- Product Management
- Category Filtering
- Shopping Cart
- Order Management
- Razorpay Payment Gateway
- Cloudinary Image Upload
- Admin Dashboard
- Responsive Design
- MongoDB Database

---

# 🛠 Tech Stack

## Frontend
- React.js
- Redux Toolkit
- React Router DOM
- Axios
- HTML5
- CSS3
- JavaScript

## Backend
- Node.js
- Express.js
- MongoDB

## Services
- Razorpay
- Cloudinary
